PSKOV is a simple but brilliant static site generator, and here's why:
Functionality Technical details Meaning
Generates static site Static site is simply a bunch of HTML files You can host static site anywhere, even preview locally, no server required
Runs inside your web browser PSKOV is a JavaScript application Any desktop with a web browser should be able to run PSKOV
Supports decades old operating systems PSKOV uses ECMAScript 5 (2009), which is supported by old web browsers Microsoft Windows 2000 with Firefox 12 is enough to run PSKOV. Only update when you want, not them
PSKOV is designed to work for decades ECMAScript 5 is a subset of later ECMAScript versions, i.e., when we use ECMAScript 5, we also use ECMAScript 6 but restrict ourselves to ECMAScript 5 functionality We expect web browsers to support PSKOV for at least a decade. Prove us wrong in 2030!
PSKOV is an example of so-called "Client side web" PSKOV is strictly a client side JavaScript application PSKOV does not send your data anywhere, you own PSKOV, you don't pay the bills, you juse use the tool
